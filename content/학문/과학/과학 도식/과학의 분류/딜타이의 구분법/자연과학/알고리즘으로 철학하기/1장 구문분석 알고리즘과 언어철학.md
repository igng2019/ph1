# 패턴 인식 알고리즘과 게슈탈트 이론

  

게슈탈트는 20세기에 시작된 심리학 이론으로 전체는 부분의 합보다 크다는 원칙으로 개별 요소가 아니라 전체 맥락 속에서 관계를 파악하고 구조화하는 것입니다. 근접성, 유사성 폐쇄성과 같은 속성은 인간 지각이 복잡한 정보를 단순화하고, 이를 통해 전체적인 의미를 구성하는 과정을 설명할 때 필요한 원칙입니다.

패턴 인식 알고리즘은 데이터를 체계적으로 분석하고 유의미한 패턴을 찾아내기 위해 설계된 기술로 게슈탈트와 마찬가지로 복잡한 정보를 구조적으로 이해하기 위한 신경망 기술입니다.

  

유사성은 인간이 색상,크기,모양이 비슷한 객체를 묶는 지각 과정을 설명할 때 사용하는 원칙입니다.

이는 합성곱 신경망인 cnn의 작동방식과 비슷하지요.

  

후설의 현상학은 인간 경험의 본질을 사유하면서 지각과 의식 간의 관계를 분석합니다 의식은 항상 어떤 대상에 대한 의식을 하는데 후설은 이를 지향성이라고 명명했죠, 의식이 정보를 받아들이는 수동적 과정이 아니라, 대상을 향하고 의미를 구성하는 능동적 활동이라는 의미입니다. 이런 지향성 개념은 인간 지각의 맥락적 특성을 강조합니다.
