# 모리스 블랑쇼의 저작과 사유의 돌파: 부재와 타자성을 향한 여정

모리스 블랑쇼(Maurice Blanchot)의 사유는 고정된 체계를 거부하며, 문학적 글쓰기와 철학적 사유의 경계에서 끊임없이 자신을 해체하고 재구성하는 과정을 보여줍니다. 그의 각 저작은 이전 시기의 사유가 봉착했던 한계를 돌파하기 위한 치열한 노력이었으며, 이는 주체 중심의 형이상학에서 벗어나 '중성(le neutre)'과 '타자성'의 영역으로 나아가는 이행의 역사입니다.

## 1. 초기 소설과 존재의 익명성 (1940년대)

블랑쇼의 초기 작업은 전통적인 서사 구조와 인물 중심의 소설 작법을 근본적으로 뒤흔드는 데 집중되었습니다. 이 시기의 핵심적인 돌파는 주체의 능동적 죽음에서 익명적인 존재의 밤으로의 이행입니다.

|   |   |   |   |
|---|---|---|---|
|주요 저작|출간 연도|이전의 문제 및 한계|사유의 돌파 지점|
|어둠의 토마 (Thomas l'Obscur)|1941|고전적 소설의 재현 체계와 명확한 자아를 가진 주인공의 서사.|'일리아(il y a)'의 경험을 통해 주체성이 붕괴되고 존재의 익명성 속에 매몰되는 과정을 묘사.|
|사형선고 (L'Arrêt de mort)|1948|죽음을 삶의 완성이나 의미 있는 종결로 보는 실존주의적 관점.|죽음이 결코 도래하지 않는, 끝없는 기다림과 불가능한 죽음의 경험을 서사화.|

블랑쇼는 《어둠의 토마》를 통해 하이데거적 '존재'가 아닌, 의미도 주체도 없는 순수한 존재의 지속인 '일리아'를 포착해냈습니다. 이는 주체가 세상을 인식하고 지배하는 능동성을 상실하고, 오히려 존재의 거대한 밤에 압도당하는 수동성을 드러낸 결정적 돌파였습니다.

## 2. 문학 이론의 정립과 탈작업의 사유 (1950년대)

1950년대에 들어 블랑쇼는 문학의 본질에 대한 이론적 성찰을 심화시킵니다. 이 시기의 돌파는 문학을 완성된 작품(Work)으로 보는 관점에서 탈작업(Worklessness, désoeuvrement)의 과정으로 보는 관점으로의 전환입니다.

|   |   |   |   |
|---|---|---|---|
|주요 저작|출간 연도|이전의 문제 및 한계|사유의 돌파 지점|
|불의 몫 (La Part du feu)|1949|문학을 지식의 전달이나 도덕적 교훈의 도구로 보는 도구적 관점.|언어가 사물을 죽임으로써 의미를 생성한다는 '부정성'의 논리를 문학 비평에 도입.|
|문학의 공간 (L'Espace littéraire)|1955|작가의 의도나 작품의 완결성을 중시하는 전통적 비평.|오르페우스 신화를 통해 '작품의 부재'를 유지하는 것이 문학의 진정한 본질임을 역설.|

《문학의 공간》에서 제시된 '오르페우스의 시선'은 작품을 완성하려는 의지가 오히려 작품의 근원(에우리디케)을 파괴한다는 역설을 보여줍니다. 블랑쇼는 이를 통해 문학이 도달해야 할 지점은 완성된 텍스트가 아니라, 그 텍스트를 가능하게 하면서도 동시에 해체하는 '부재의 공간'임을 천명했습니다.

## 3. 중성과 파편적 글쓰기의 도래 (1960-70년대)

1960년대 이후 블랑쇼의 사유는 레비나스의 타자 철학을 수용하며 윤리적, 정치적 차원으로 확장됩니다. 이 시기의 돌파는 동일성의 논리를 넘어선 파편적이고 중성적인 관계의 모색입니다.

|   |   |   |   |
|---|---|---|---|
|주요 저작|출간 연도|이전의 문제 및 한계|사유의 돌파 지점|
|미래의 책 (Le Livre à venir)|1959|책의 물리적 형태와 선형적 구조에 갇힌 사유.|미래의 책은 고정된 형태가 아니라 끊임없이 변화하고 흩어지는 '글쓰기' 자체임을 제시.|
|무한한 대화 (L'Entretien infini)|1969|합의와 이해를 목적으로 하는 변증법적 대화.|타자와의 절대적 거리를 유지하며, 결코 통합되지 않는 '중성(le neutre)'의 대화를 제안.|
|저편으로의 발걸음 (Le Pas au-delà)|1973|역사적 진보와 목적론적 시간관.|'발걸음(pas)'과 '부정(pas)'의 이중 의미를 통해, 경계를 넘어서면서도 머무는 파편적 사유를 실천.|

《무한한 대화》는 블랑쇼 사유의 정점으로, 헤겔적 변증법이 가진 전체주의적 위험을 경계합니다. 그는 대화를 두 주체의 결합이 아니라, 그들 사이의 '심연'을 보존하는 행위로 재정의함으로써, 타자를 자신의 체계로 환원하지 않는 진정한 윤리적 관계를 정초했습니다.

## 4. 재난과 불가능한 공동체 (1980년대 이후)

말년의 블랑쇼는 역사적 비극과 공동체의 문제를 다룹니다. 이 시기의 돌파는 주체의 통제권을 완전히 상실한 재난의 상태를 사유의 출발점으로 삼는 것입니다.

|   |   |   |   |
|---|---|---|---|
|주요 저작|출간 연도|이전의 문제 및 한계|사유의 돌파 지점|
|재난의 글쓰기 (L'Écriture du désastre)|1980|사건을 경험하고 기록하는 주체의 능동성.|재난은 이미 일어났으나 결코 현존하지 않는 사건이며, 글쓰기는 그 흔적을 따라가는 수동적 행위임을 강조.|
|밝힐 수 없는 공동체 (La Communauté inavouable)|1983|공통의 속성이나 이념에 기반한 제도적 공동체.|죽음과 부재를 공유하며, 결코 하나로 묶일 수 없는 '불가능한 공동체'의 형상을 제시.|

《재난의 글쓰기》에서 '재난'은 주체가 감당할 수 없는 타자성의 극단적 형태입니다. 블랑쇼는 파편적 글쓰기를 통해 재난의 충격을 그대로 보존하며, 어떤 체계로도 포섭되지 않는 사유의 불가능성을 증언했습니다. 이는 현대 철학이 마주한 거대한 한계를 글쓰기라는 실천을 통해 돌파하려는 마지막 노력이었습니다.

## 결론: 끊임없는 이행으로서의 사유

모리스 블랑쇼의 저작들은 하나의 완성된 체계를 향해 나아가지 않습니다. 오히려 각 저작은 이전의 사유가 안주하려 했던 지점을 파괴하고, 더 깊은 밤, 더 먼 타자성을 향해 나아가는 '발걸음'입니다. 그의 사유는 우리에게 문학이, 그리고 철학이 어떻게 자신의 한계를 돌파하며 '말할 수 없는 것'에 응답할 수 있는지를 보여주는 가장 정교한 지도입니다.
